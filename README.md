# Mini Katalog Uygulaması

Bu proje, ürünlerin veya içeriklerin listelendiği, kullanıcı dostu bir arayüze sahip mobil bir katalog uygulamasıdır.

## 📝 Proje Açıklaması
Bu uygulama, belirli bir ürün grubunu kategorize etmek, detaylarını görüntülemek ve kullanıcılara interaktif bir katalog deneyimi sunmak amacıyla geliştirilmiştir. Proje, temiz kod yapısı ve modern Flutter UI bileşenleri kullanılarak oluşturulmuştur.

## 🚀 Kullanılan Teknolojiler ve Sürüm
* **Framework:** Flutter
* **Dil:** Dart
* **Flutter Sürümü:** 3.41.6

## 🛠️ Çalıştırma Adımları

Projeyi kendi yerel makinenizde sorunsuz bir şekilde çalıştırmak için aşağıdaki adımları takip edebilirsiniz:

1. **Projeyi bilgisayarınıza klonlayın:**
git clone [https://github.com/0nurTURAN/Mini-katalog-uygulamasi.git](https://github.com/0nurTURAN/Mini-katalog-uygulamasi.git)
Proje dizinine gidin:

cd Mini-katalog-uygulamasi

Gerekli paketleri ve bağımlılıkları yükleyin:

flutter pub get

Uygulamayı bir emülatör veya gerçek cihaz üzerinde başlatın:

flutter run


Bu proje Onur TURAN tarafından geliştirilmiştir.
